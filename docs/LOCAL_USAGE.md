# 本機使用說明（詳細版）

本文件提供「一般使用者」與「IT/教學端」在 Windows 本機使用 `SubtitleTool.exe` 的完整指引。

## A. 建議環境

- 作業系統：Windows 10/11（64-bit）
- CPU：可用（速度取決於模型大小與檔案長度）
- GPU（選用）：NVIDIA GPU + 驅動（是否可用依套件與驅動相容性）
- 網路：第一次使用某模型時需要下載模型檔（之後可離線）
- ffmpeg：建議安裝並加入 PATH（提升相容性）

## B. 安裝 ffmpeg（建議）

1. 下載 ffmpeg（Windows build）
2. 解壓縮後把 `bin` 目錄加入系統 PATH
3. 重新開啟命令列，確認：

```powershell
ffmpeg -version
```

## C. 使用步驟（最常見）

1. 執行 `SubtitleTool.exe`
2. 先按「檢查環境」
3. 按「選擇檔案」挑音訊/影片
4. 選擇模型（第一次會下載）
5. 按「開始辨識」
6. 完成後按「另存 SRT」

## D. 模型選擇建議

- 先用 `base` 或 `small` 測流程
- 需要更高準確率再提升到 `medium` 或 `large-v3`
- 檔案很長時，模型越大耗時越久

## E. 字幕切段規則（fine/standard/loose）

字幕切段的目的：
- 每行不要太長（可讀性）
- 每段停留時間合理（跟得上）
- 在標點符號處更自然斷句

如果你覺得字幕太碎：改用 `loose`  
如果你覺得字幕太長：改用 `fine`

## F. 常見錯誤與排除

### F.1 提示找不到 `silero_vad_v6.onnx`

請更新到本 repo 最新版 exe（此版本已把 VAD 模型資源納入打包）。

### F.2 影片打不開 / 沒有聲音 / 報解碼錯誤

優先安裝 ffmpeg 並加入 PATH，再重試。

### F.3 公司網路擋模型下載

解法：
- 先在可連線環境把模型跑過一次（完成下載）
- 再把快取搬到目標機器（或同帳號下直接復用快取）
